# Rails-Simple-Chat
This is rails project to create simple chat. You are expected to have redis-server available for turbo-stream.

# Start local
1. Turn on redis-server
2. cmd rails s

# System dependencies
* ruby 3.1.2p20
* Rails 7.0.3
* Redis 5.0.7
* Ubuntu 20.04.4 LTS
